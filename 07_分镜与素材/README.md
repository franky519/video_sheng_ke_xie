本目录是生产链路的核心中间产物层。生产流程：

```
01_文字稿  ──┐
             ├→  AI  →  分镜脚本/  →  AI  →  素材清单/  →  AI+人  →  assets/
02_拆解样本 ─┘
04_Prompt ───┘
```

## 目录结构

```
07_分镜与素材/
├── 分镜脚本/          AI 基于文字稿 + Few-Shot 样本生成的逐秒分镜
├── 素材清单/          从分镜脚本提取的结构化素材待办清单
├── 素材候选池/        自动找图的中间产物（每个素材6-10张候选卡片）
└── assets/            最终选定的素材文件 + 素材台账
```

## 素材生命周期

```
[TODO] → AI 改写搜索词 → 并行检索图库 → AI 去重过滤
      → [CANDIDATES_READY] 生成候选卡片
      → 人选中1张
      → [SELECTED] 下载到 assets/ASSET_xx_xx.png
      → [DONE] 更新素材台账
```

## 文件命名

- 分镜脚本：`YYYY-MM-DD_HH-MM_豆包科普视频_完整分镜脚本_vN.md`
- 素材清单：`YYYY-MM-DD_HH-MM_豆包科普视频_素材提取清单_vN.md`
- 素材文件：`ASSET_[镜头序号]_[图层序号].png`（如 ASSET_01_02.png）
- 素材台账：`assets/asset_ledger.json`
