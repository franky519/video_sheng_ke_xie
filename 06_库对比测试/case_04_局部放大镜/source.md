# Case 4 — 局部放大镜（圆形 ClipPath + 内部放大 + 抖动）

## 场景描述

专家回复卡片右下角出现一个圆形放大镜，对准附录小字区域。放大镜内部将文字局部放大 2.5 倍，镜片中心做 ±2px 正弦抖动，模拟手持放大镜的轻微晃动感。

来源：
- `05_写代码做视频的工程/_归档_旧版方案/2026-06-03_09-20_豆包科普视频_0_30s图层运动工程调整与待办脚本.md` 第49-50行

## 物理参数

| 参数 | 值 |
|------|-----|
| 动画类型 | 圆形 ClipPath 遮罩 + 内部 Scale + 正弦抖动 |
| 裁剪形状 | `circle(120px)` |
| 内部缩放 | `scale(2.5)` |
| 抖动幅度 | ±2px 正弦抖动 |
| 定位 | 右下角，TranslateX: 35%, TranslateY: 25% |
| 出现时间 | 0.0s - 0.8s（弹性弹出） |
| 持续抖动 | 0.8s - 3.0s |

## 画面内容

| 元素 | 规格 |
|------|------|
| 背景 | 纯色 RGB(30, 32, 35) |
| 主卡片 | 深灰圆角矩形，800×400px，背景 RGB(50,53,58)，圆角 12px，居中 |
| 卡片内容 | 上半部分是大号白色标题和 3 行正文文字占位条；下半部分（底部 80px）是小字号附录文字 |
| 放大镜 | 圆形，半径 120px，位置在卡片右下角附录文字上方 |
| 放大镜边框 | 白色发光圆环 2px |
| 放大镜内部 | 与卡片同源内容，但 `scale(2.5)` + 反向定位补偿 |
| 抖动 | 使用 `Math.sin` × 2px 偏移 |
