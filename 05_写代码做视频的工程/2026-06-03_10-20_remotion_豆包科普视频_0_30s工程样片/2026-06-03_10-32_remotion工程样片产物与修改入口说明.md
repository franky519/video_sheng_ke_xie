## 工程入口怎么找

| 你想改什么 | 进入哪个文件 |
| :--- | :--- |
| 调整某段时间轴、Spring、Opacity、Translate、Blur、ClipPath | `src/composition/DoubaoScience30.tsx` |
| 调整 0~30 秒四段的起止时间、字幕 | `src/lib/timeline.ts` |
| 调整弹性、线性、平滑插值函数 | `src/lib/motion.ts` |
| 调整手机、提问卡、普通回答卡、专家卡、放大镜、提示词卡 | `src/components/` |
| 调整字体、颜色、间距、卡片尺寸、背景网格 | `src/styles.css` |
| 替换真实截图或素材状态 | `asset-manifest.json` 和 `public/assets/` |

如果要改第 9 秒附近的普通回答卡，先看 `src/composition/DoubaoScience30.tsx` 里的 `PlainAnswerScene`，再看 `src/components/Cards.tsx` 里的 `PlainAnswerCard` 和 `src/styles.css` 里的 `.plain-answer-*` 样式。

如果要改第 22 秒附近的放大镜，先看 `ExpertAnswerScene` 里的 `magnifierScale`、`jitterX`、`jitterY` 和 `.magnifier-wrap`，再看 `src/components/Cards.tsx` 里的 `Magnifier`。

## 帧号换算

本工程是 24fps，所以帧号等于秒数乘以 24。

| 时间 | 帧号 |
| :--- | :--- |
| 2 秒 | 48 |
| 9 秒 | 216 |
| 16 秒 | 384 |
| 22 秒 | 528 |
| 28 秒 | 672 |

要导出任意单帧，可以运行：

```bash
npx remotion still src/index.tsx DoubaoScience30 out/frame_custom.png --frame=216 --overwrite
```

要在浏览器里拖动时间线看任意帧，可以运行：

```bash
npm run studio
```
