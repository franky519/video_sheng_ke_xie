# Case 8 — 贝塞尔路径运动

## 场景描述

一个圆形元素沿 cubic-bezier(0.25, 0.1, 0.25, 1.0) 曲线路径从画面左下角移动到右上角。用于验证各库对贝塞尔曲线路径动画的支持精度。

来源：
- `04_教AI拉片的提示词/2026-06-09_17-30_Gemini物理拉片分析prompt_v2.txt` 第40-43行（Bezier参数规范）

## 物理参数

| 参数 | 值 |
|------|-----|
| 动画类型 | 沿贝塞尔曲线路径移动 |
| 路径曲线 | cubic-bezier(0.25, 0.1, 0.25, 1.0) 形状 |
| 路径起止 | 左下 (10%, 85%) → 右上 (90%, 15%) |
| 控制点 | 根据 bezier 参数推导 |
| 持续时间 | 2.0 秒 |
| 缓动 | 沿路径匀速 |

## 画面内容

| 元素 | 规格 |
|------|------|
| 背景 | 纯色 RGB(30, 32, 35) |
| 路径可视 | 半透明灰色虚线，展示完整路径 |
| 运动元素 | 绿色圆点，半径 20px，带发光 |
| 运动方式 | 从起点沿贝塞尔路径平滑移动到终点 |
