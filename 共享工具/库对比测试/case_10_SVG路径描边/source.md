# Case 10 — SVG 路径描边（Stroke Dasharray 动画）

## 场景描述

产品层、路由层、模型层三层结构图依次出现，连接箭头逐段点亮。用于解释"AI 模型推理的分层架构"。

来源：
- `01_文字稿与原始素材/_归档_旧版素材表/2026-05-31_13-06_豆包科普视频逐镜头素材实现表.csv` S08行

## 物理参数

| 参数 | 值 |
|------|-----|
| 动画类型 | SVG stroke-dasharray 描边 |
| 层数 | 3 层（产品层、路由层、模型层） |
| 每层出现 | 0.5-0.8s |
| 总时长 | 约 3 秒（三层 + 箭头） |
| 层间间隔 | 0.6s |
| 箭头点亮 | 跟随路径方向逐段显示 |

## 画面内容

| 元素 | 规格 |
|------|------|
| 背景 | 纯色 RGB(30, 32, 35) |
| 矩形框 | 3 个横向排列圆角矩形，白色描边，代表三层 |
| 标签 | 框内文字："产品层 / 路由层 / 模型层" |
| 箭头 | SVG 箭头路径连接各层 |
| 描边动画 | stroke-dasharray 从 0 到全长 |
