# Case 3 — ClipPath 水平擦除（高亮色条）

## 场景描述

深色专家回复卡片上，绿色荧光条和橙色荧光条依次从左向右水平展开。用于视觉引导观众逐条阅读关键信息，模拟"荧光笔划重点"的效果。

来源：
- `视频项目/01_为什么他的豆包更聪明/05_渲染工程/_归档_旧版方案/2026-06-03_09-20_豆包科普视频_0_30s图层运动工程调整与待办脚本.md` 第49行

## 物理参数

| 参数 | 值 |
|------|-----|
| 动画类型 | ClipPath 水平擦除 |
| 绿色条 ClipPath | `inset(0% 100% 0% 0%)` → `inset(0% 0% 0% 0%)` |
| 橙色条 ClipPath | 同上（各自独立） |
| 绿色条时间 | 0.0s - 1.0s |
| 橙色条时间 | 1.0s - 2.0s（延迟 1s） |
| 持续时间 | 每个色条 1.0s |
| 缓动函数 | ease-out（先快后慢，有"划出"的流畅感） |

## 画面内容

| 元素 | 规格 |
|------|------|
| 背景 | 纯色 RGB(30, 32, 35) |
| 卡片 | 深灰圆角矩形，800×360px，背景 RGB(50,53,58)，圆角 12px，居中 |
| 卡片内容 | 3 行白色文字占位条，行高充足，间距 40px |
| 绿色高亮条 | 覆盖在第 1 行文字上方，半透明绿色 RGB(0,200,80)，高度 26px，初始完全隐藏（右侧裁掉） |
| 橙色高亮条 | 覆盖在第 2 行文字上方，半透明橙色 RGB(255,160,20)，高度 26px，初始完全隐藏 |
| 擦除方向 | 从左到右（右边收起 → 右边展开） |
