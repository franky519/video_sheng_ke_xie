# Case 9 — 数字滚动动画

## 场景描述

一个成本阶梯图场景中，数字从 0 快速滚动到目标值（如从 0 到 $0.0035），柱子同时从 0 升起。用于展示"模型推理成本差异"的关键数据。

来源：
- `_归档_/视频项目/01_为什么他的豆包更聪明/旧版素材表/2026-05-31_13-06_豆包科普视频逐镜头素材实现表.csv` S14行

## 物理参数

| 参数 | 值 |
|------|-----|
| 动画类型 | 数字滚动 |
| 滚动时长 | 12-18 帧（0.4-0.6s @30fps） |
| 起始值 | 0 |
| 目标值 | 0.0035 |
| 柱子升起 | 0.6-1.0s，从 0 到目标高度 |
| 缓动 | ease-out（先快后慢） |

## 画面内容

| 元素 | 规格 |
|------|------|
| 背景 | 纯色 RGB(30, 32, 35) |
| 数值显示 | 大号数字 "$0.0035"，居中上方，白色 |
| 数字动画 | 从 0 快速递增到目标值 |
| 柱子 | 竖条形，宽 80px，从底部升起，绿色 RGB(0,200,80) |
