# Case 5 — 错层入场（Staggered 卡片入场）

## 场景描述

右侧专家窗口内，三条风险提示卡片依次错层出现（staggered），每条间隔约 0.8-1.2 秒。用于制造"信息逐步展开"的节奏感，引导观众逐条阅读。

## 物理参数

| 参数 | 值 |
|------|-----|
| 动画类型 | 错层入场（Staggered） |
| 卡片数量 | 3 条 |
| 间隔时间 | 0.8s（每条之间的延迟） |
| 入场方式 | TranslateX 120% → 0%（从右侧滑入） |
| 旋转 | Rotate 8deg → 0deg（轻微旋转落下） |
| 持续时间 | 每条动画约 0.6s |
| 缓动 | ease-out（先快后慢） |

| 元素 | 规格 |
|------|------|
| 背景 | 纯色 RGB(30, 32, 35) |
| 容器 | 右侧专家窗口区域，深灰背景 |
| 卡片 | 3 条横向风险提示卡片，每条 680×64px，背景 RGB(60,65,70)，圆角 8px |
| 卡片文字 | 3 条风险标题："免责条款容易漏看"、"赔付条件差异大"、"附录小字需注意" |
| 布局 | 垂直排列，间距 20px，从右侧向左滑入，带轻微旋转 |
