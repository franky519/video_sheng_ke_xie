# Case 2 — 弹性弹出（Spring Scale）

## 场景描述

两个手机造型的卡片从画面两侧弹性弹出。左侧"快速模式"手机先弹出（0-1秒），右侧"专家模式"手机延迟1秒后弹出（1-2秒）。这是视频开头的核心画面，类比"同一问题两种豆包"的视觉反差。

来源：
- `05_写代码做视频的工程/_归档_旧版方案/2026-06-03_09-20_豆包科普视频_0_30s图层运动工程调整与待办脚本.md` 第47行

## 物理参数

| 参数 | 值 |
|------|-----|
| 动画类型 | Scale 弹性弹出 |
| Scale 变化 | 0% → 100%（spring） |
| Spring Damping | 15 |
| Spring Mass | 0.6 |
| 左侧卡片 | 0.0s - 1.0s |
| 右侧卡片 | 1.0s - 2.0s（延迟 1s） |
| 持续时间 | 每个卡片动画约 1 秒后稳定 |

## 画面内容

| 元素 | 规格 |
|------|------|
| 背景 | 纯色 RGB(30, 32, 35) |
| 左侧卡片 | 手机造型，280×500px，圆角 16px，背景 RGB(50,53,58)，居中偏左（TranslateX: -20%） |
| 右侧卡片 | 手机造型，280×500px，圆角 16px，背景 RGB(50,53,58)，居中偏右（TranslateX: 20%） |
| 卡片标题 | 左侧"快速模式"，右侧"专家模式"，白色文字 20px |
| 卡片内容 | 4-5 行灰色文字占位条，模拟聊天回答 |
| 初始状态 | 两张卡 Scale 均为 0%（不可见），逐一弹性弹出至 100% |
