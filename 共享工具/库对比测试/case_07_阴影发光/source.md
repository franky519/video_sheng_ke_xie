# Case 7 — 阴影发光（Drop Shadow Glow + Spring Scale）

## 场景描述

画面中央出现一个红色问号，Scale 从 0 弹到 120% 再稳定到 100%（spring），同时 drop-shadow 发光从 0 增长到 30px。用于强调"观众的第一反应：是不是我不会提问？"

## 物理参数

| 参数 | 值 |
|------|-----|
| 动画类型 | Spring Scale + Drop Shadow Glow |
| Scale 变化 | 0% → 120% → 100%（spring） |
| Spring Damping | 10 |
| Spring Mass | 0.5 |
| Drop Shadow | 0px → 30px |
| 出现时间 | 0.0s - 0.5s（缩放段） |
| 稳定后微呼吸 | scale ±1.5% 正弦微动 |

## 画面内容

| 元素 | 规格 |
|------|------|
| 背景 | 纯色 RGB(30, 32, 35) |
| 问号 | 红色大号问号 `?`，font-size 120px，红色 RGB(255,60,60)，居中 |
| 发光 | 红色 drop-shadow，半径从 0 到 30px |
| 动画 | 弹性弹出 + 持续微呼吸 + 发光渐强 |
