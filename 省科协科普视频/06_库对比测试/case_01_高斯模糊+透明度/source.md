# Case 1 — 高斯模糊 + 透明度叠加退场

## 场景描述

左侧聊天窗口显示了一段平庸的保险条款回答（大段灰色文字）。在 1.5 秒内，这个窗口需要同时经历模糊和变透明的效果，视觉重心自然转移到右侧的专家窗口。

## 物理参数

| 参数 | 值 |
|------|-----|
| 时长 | 1.5 秒 |
| Opacity 变化 | 100% → 30%（线性） |
| Blur 变化 | 0px → 18px（线性） |
| 缓动函数 | linear（匀速） |

## 画面内容

| 元素 | 规格 |
|------|------|
| 背景 | 纯色 RGB(30, 32, 35) |
| 卡片 | 灰色圆角矩形，RGB(50, 53, 58)，圆角 12px，带轻微投影 |
| 卡片尺寸 | 宽 800px × 高 300px，居中放置 |
| 卡片内容 | 4 行中文占位文字，白色/浅灰，模拟保险条款回答 |
| 其他元素 | 画面右上角保留一个较小的对比窗口（不作为动画目标），以模拟"视觉重心转移"的上下文 |
