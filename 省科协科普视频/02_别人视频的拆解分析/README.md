## 质量标准

仅 gemini-3.5-flash + v2 prompt 的分段拉片结果作为有效 Few-Shot 样本。

v2 prompt 的核心改进：删除库参考列；新增设计意图（运动→叙事目的）；素材打标分类；静态层单独声明。

旧版结果（v1 prompt、gemini-2.5-flash、早期抽帧分析）已归入 `_归档_旧版结果/` 或保留在各自视频目录内作为历史参考。
