# 手写参考版：CODE_DATA 排名动画

> 测试时间：2026-06-24

## 测试目的

验证纯 Remotion `interpolate()` + `spring()` 能否驱动数据动画。数据硬编码为 TS 数组，不用外部库。

## 工具链

Remotion `useCurrentFrame()` + `interpolate()` + `spring()`，纯帧驱动。
