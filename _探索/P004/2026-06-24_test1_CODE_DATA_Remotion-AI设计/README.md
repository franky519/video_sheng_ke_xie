# CODE_DATA 视觉导演 — 设计决策记录

> 创建时间：2026-06-24
> 旁白场景：Chatbot Arena 排名从 #8 跌至 #35

## 情绪判断

**震撼 + 紧迫感。** 旁白先铺垫"曾排进前十、唯一国产模型"的荣耀，紧接着"仅仅两个月后掉到第 35 位"制造巨大落差。这不是愤怒或幽默，而是用数据崩塌的速度感让观众感受"AI 竞争有多惨烈"。画面需要突出：从金灿灿的巅峰急速坠落到冷红色的平庸。

## 画面描述

深色科技感背景（`#080C14`），80px 网格线铺底。顶部浮现 "Chatbot Arena 全球盲选评测 · 文本榜排名" 标题。左侧有模糊的半透明排行榜列表（模拟竞品模型名称），增强"排名下跌"的空间参照。

中央一个带发光边框的深色卡片（`#111827`），内含巨大的排名数字。初始显示金色 "#8"，外围金色径向光晕，卡片下方标签显示 "全球第一梯队 TOP 10" 和时间上下文 "2026 年 3-4 月 · 唯一挤进第一梯队的国产模型"。

约 1.7 秒后排名开始坠落：数字从 8 快速计数到 35，光晕颜色从金色 `#F5A623` 线性过渡到冷红色 `#E84040`。卡片整体向下滑动约 200px，微缩至 82%，并伴随水平微震（`sin` 波浪模拟跌落冲击）。左侧排行榜列表同步快速滚动，制造"被竞品超过"的视觉感。

底部有一根金色→红色渐变的进度条，从 #10 端滑向 #35 端，指示盘上带发光圆点。最终画面定格在 "#35"，底部浮现红色结论文字 "AI 模型竞争：两个月从巅峰到平庸"。

## 设计决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 核心动效 | 排名数字计数 + 卡片下滑 + 颜色渐变 | 三者叠加产生"从荣耀跌落到平庸"的复合感受，比单纯数字变化更有戏剧性 |
| 颜色方案 | 金色 `#F5A623` → 红色 `#E84040` | 金色代表荣耀/稀有（top 10），红色代表警示/衰落，心理暗示强烈 |
| 微震动 | `sin(frame * 0.7)` 水平位移 0-5px | 模拟排名骤降时的"撞击感"，差评君风格中常用的小技巧增强数据冲击力 |
| 左侧排行榜 | 模糊半透明竞品列表 + 快速滚动 | 提供空间参照——不是模型主动变差，而是被竞品超越 |
| 进度条 | Linear 橙色→红色渐变 + 发光指示点 | 把"两个月"的时间流逝具象化，与排名数字同步推进 |
| 不使用外部库 | 纯 Remotion `interpolate` + `spring` + `AbsoluteFill` | 满足约束，所有动效通过物理字典原语实现（TranslateY/Scale/Opacity） |
| 卡片容器 | `CARD_BG` 深色半透明 + 彩色边框 + `boxShadow` 光晕 | 模拟 `[CODE]` 弹窗/文字卡片原语，边框颜色随排名联动变色 |
