# 素材清单

创建时间：2026-06-11 20:00（北京时间）
最后更新时间：2026-06-15 17:16（北京时间）
更新次数：2（目录体系重构）

本目录管理视频项目的素材全生命周期：从分镜提取的素材清单，到候选池筛选，到最终素材。

```
00_文稿  ──┐
           ├→  AI  →  03_分镜脚本/  →  AI  →  04_素材清单/  →  AI+人  →  assets/
标杆视频库 ─┘
共享工具/提示词 ─┘
```

## 目录结构

```
04_素材清单/
├── *.json / *.md      结构化素材清单文件
├── 素材候选池/        自动找图的中间产物（每个素材6-10张候选卡片）
└── assets/            最终选定的素材文件 + 素材台账（source of truth）
```

## 素材生命周期

```
[TODO] → AI 改写搜索词 → 并行检索图库 → AI 去重过滤
      → [CANDIDATES_READY] 生成候选卡片
      → 人选中1张
      → [SELECTED] 下载到 assets/ASSET_xx_xx.png
      → [DONE] 更新素材台账
```

## 文件命名

- 素材清单：`YYYY-MM-DD_HH-MM_豆包科普_0-30s_素材清单.{json,md}`
- 素材文件：`ASSET_[镜头序号]_[图层序号].png`（如 ASSET_01_02.png）
- 素材台账：`assets/asset_ledger.json`
